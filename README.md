# activistsearch
Goal is to find high-potential activist investment opportunities. 

System scores activist opportunities with criteria based on analyses of 19 of the most successful activist campaigns of the last 2 decades. Factors: 

1. Quantifiable, Peer-Verifiable Performance Gap
2. Credible, Executable Fix
3. Balance Sheet / Capital Allocation Slack 
4. Live Catalyst / Governance Vulnerability
5. Structural Feasibility / Absence of Red Flags

Input: desired tickers
Output: score on each, research reports, playbooks.
